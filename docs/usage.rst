=====
Usage
=====

To use asciju in a project::

    import asciju

Convert your image into ASCII art::
 
    from asciju import convert_img_ascii

    print(convert_img_ascii('yourimage.jpeg','output.txt'))

Run the python script::

    python3 nameofthescript.py
